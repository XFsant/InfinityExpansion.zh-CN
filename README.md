# Infinity Expansion

此插件需要Slimefun
通过添加“无尽锭”以及其他的机器来增加游戏的乐趣

## 中文版下载

https://github.com/baoad/InfinityExpansion/releases



## 错误/建议

请到issus板块来发问错误/建议


## 需求

Minecraft版本: 1.16+ (建议使用Paper)

Slimefun版本: DEV 925+

推荐您用StarWishsama汉化的最新的Slimefun插件
